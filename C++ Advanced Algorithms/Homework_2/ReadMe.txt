main.cpp:
	Brute force algorithm that finds whether a possible assignment of squares satisfies our sub
	set rules.

	To compile: g++ main.cpp

	To run: ./a.out (#of squares) (#of sets) 
		Ex: ./a.out 15 4

prune.cpp:
	Utilizes code tuning and pruning to speed up computation

	To compile: g++ prune.cpp

	To run: ./a.out (#of squares) (#of sets)
		Ex: ./a.out 15 4
