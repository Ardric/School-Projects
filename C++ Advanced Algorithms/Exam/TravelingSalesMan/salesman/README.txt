Daniel Lowdermilk - Problem 0 Traveling Salesman

How it works:
	Reads graph from file into adjacency matrix.
	Generates all possible subsets of vertices for the graph
	Utilitzes dynamic programming to generate shortest path
	Returns shortest path and its weight on completion
